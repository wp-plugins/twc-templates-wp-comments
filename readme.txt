Plugin for creating tree wordpress comments with spam protection and unfolding responses

== Description ==

Wordpress plugin to create a tree of comments on the basis of standard functions.

Plugin TWC-Templates-WP-Comments loads the template to display the comments instead of the standard WP or regular file comments.php

The plugin has a built-in spam protection, which eliminates the automatic comment spam to 99%.

== Installation ==

To call the plugin, simply insert a standard function comments_template(), which is normally present in any subject.
If this function is already installed in the theme file single.php or page.php, you need activate plugin TWC-Templates-WP-Comments.
